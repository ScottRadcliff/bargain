# Bargain

Bargain is a Markdown parsing library for Elixir. It allows you to take markdown and return valid HTML.


### Features

**Parse the following**

- [ ] Heading
- [ ] Paragraphs
- [ ] Links
- [ ] Italics
- [ ] Bold
- [ ] Block Quote
- [ ] Inline Code
- [ ] Code Block
- [ ] Horizontal Rules
- [ ] Automatic Links
- [ ] Images
