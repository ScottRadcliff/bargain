# Bargain

Bargain is a Markdown parsing library for Elixir. It allows you to take markdown and return valid HTML.


### Features

**Parse the following**

- [x] Heading
- [x] Paragraphs
- [x] Links
- [x] Italics
- [x] Bold
- [ ] Block Quote
- [ ] Inline Code
- [ ] Code Block
- [ ] Horizontal Rules
- [ ] Automatic Links
- [ ] Images
